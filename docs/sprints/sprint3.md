# Sprint 3 - Implementação Visual e Sonora

### 1. Foco
O foco desta etapa é realizar a transição do protótipo cinza para uma experiência imersiva e aterrorizante. Blocos geométricos temporários serão substituídos pelos assets 3D finais, texturas e iluminação dinâmica. Além disso, o sistema de som espacial tridimensional será integrado para construir a atmosfera de horror, enquanto a lógica de programação ligará as interações do jogador às respostas validadas dos puzzles.
