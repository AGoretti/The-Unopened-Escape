#  Sprint 1 - Fundações e Game Design

---

### 1. Foco
O objetivo principal desta sprint é estabelecer as fundações estruturais e conceituais do projeto. Toda a lógica de regras dos puzzles e o fluxo de informações dos manuais serão definidos aqui. Paralelamente, a equipe irá configurar os ambientes locais do Unity e o repositório do GitHub para garantir que todos consigam trabalhar em sincronia, eliminando barreiras técnicas iniciais.
